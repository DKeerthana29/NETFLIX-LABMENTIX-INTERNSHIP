# NETFLIX-LABMENTIX-INTERNSHIP
Netflix project with Amazon Prime EDA 
Movie Data Analysis & Rating Prediction
**Project Overview**

This project focuses on Exploratory Data Analysis (EDA) and Machine Learning (ML) on a movie dataset. The goal is to analyze trends, patterns, and relationships in movie data and build a predictive model to estimate IMDb ratings.

The project combines data analysis, visualization, and predictive modeling to generate meaningful insights for decision-making in the entertainment industry.

**Objectives**
Perform data cleaning and preprocessing
Analyze movie trends using EDA techniques
Visualize data using various charts (20 visualizations)
Identify key factors affecting movie ratings
Build a machine learning model to predict IMDb scores
**Dataset**

**The dataset contains:**

Movie details (title, genre, release year, runtime)
Popularity metrics
IMDb ratings
Additional metadata

**Files used:**

titles.csv
credits.csv
**Data Preprocessing**
Removed duplicate records
Handled missing values
Selected relevant features
Merged datasets where required
Prepared clean dataset for analysis
**Exploratory Data Analysis (EDA)**
Performed 20 visualizations following:
Univariate Analysis
Bivariate Analysis
Multivariate Analysis
Key Insights:
Most movies have ratings between 5–8
Movie production increased significantly after 2000
Weak correlation between features like runtime and ratings
Popularity does not always indicate quality
**Business Insights**
Popular genres and trends can guide content creation
Ratings depend on multiple factors, not a single feature
Data-driven decisions improve success rate of movies
Marketing alone does not guarantee high ratings
**Challenges**
Missing data handling
Weak correlations between variables
Limited features affecting prediction accuracy
**Future Improvements**
Include more features (genre, cast, language)
Use advanced models (XGBoost, Gradient Boosting)
Apply feature engineering techniques
Improve model accuracy
**Technologies Used**
Python

**Conclusion**

This project demonstrates how EDA and Machine Learning can be used to analyze movie data and predict ratings. The insights gained can help in making better business decisions in the entertainment industry.
